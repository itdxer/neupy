Installation
============

Install last stable version

.. code-block:: bash

    $ pip install neupy

Install the latest development version

.. code-block:: bash

    $ pip install git+https://github.com/itdxer/neupy

Or just from source

.. code-block:: bash

    git clone https://github.com/itdxer/neupy.git
    cd neupy
    python setup.py install
